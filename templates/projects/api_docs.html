POST /api/models/{id}/predict/ Run inference

Run a prediction on the specified model. Returns prediction, confidence, label and latency.

File upload models

curl -X POST \
  {{ request.scheme }}://{{ request.get_host }}/api/models/1/predict/ \
  -F "file=@eeg_sample.csv"

Manual input models

curl -X POST \
  {{ request.scheme }}://{{ request.get_host }}/api/models/2/predict/ \
  -H "Content-Type: application/json" \
  -d '{"age": 35, "bmi": 24.5}'

Response

{
  "success": true,
  "prediction": 0.966,
  "confidence": 0.966,
  "label": "Seizure detected",
  "inference_ms": 118,
  "model_id": 1,
  "model_title": "EEG Seizure Detection"
}
